# CodeAtlas Premium Design System (Glassmorphism Edition)

## 🎨 Color Palette
- **Primary (Neon Cyan):** `#00F0FF` - Used for active nodes, primary CTA, and glow accents.
- **Secondary (Electric Violet):** `#9D00FF` - Used for AI insights, relationships, and secondary accents.
- **Background (Deep Space):** `#0A0C10` - The base layer of the void.
- **Surface (Glass):** `rgba(16, 20, 29, 0.6)` - Layered panels with transparency.
- **Border:** `rgba(255, 255, 255, 0.1)` - Ultra-thin glass edges.

## ✨ Visual Effects
- **Backdrop Blur:** `20px` - Intense frosted glass effect.
- **Outer Glow (Bloom):** `0 0 15px rgba(0, 240, 255, 0.3)` - For primary elements.
- **Rim Lighting:** `1px` top-down linear gradient borders.

## 🔡 Typography
- **Headlines:** `Space Grotesk` - Futuristic, geometric, tech-native.
- **Body:** `Inter` - High legibility for data and code.

## 📐 Shapes & Spacing
- **Corner Radius:** `24px` for cards, `12px` for buttons/inputs.
- **Grid:** 12-column layout with `24px` gutters.
- **Padding:** Generous `32px` internal padding for a premium feel.
